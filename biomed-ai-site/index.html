Technical details & code

If you're an engineer or researcher, here is what I actually implemented:

  • 3D CNN architecture: R(2+1)D-18 for temporal ultrasound videos.
  • ROI coordinate regression: train on (rx, ry) within a defined ROI.
  • K-fold training: clean train/valid split and checkpoint saving.
  • From inference to demo: checkpoint ‚Üí prediction ‚Üí red dot overlay ‚Üí output video.
